HISTÓRIA

HTML: Uma linguagem de marcação para desenvolvimento de páginas HTTP web. CSS: Uma linguagem de estilização para adição de estilos na página HTTP desenvolvida com HTML. Browser: Ferramenta conhecida como navegador, usada para buscar e visualizar páginas HTTP web. HTTP: HyperTextTransferProtocol, um protocolo cliente-servidor stateless de transferência de Hiper textos. Internet: A rede mundial que interliga diversas redes e dispositivos. TCP: Protocolo de comunicação responsável por entregar pacotes com certa garantia e configurações diversas. IP: Protocolo de endereçamento responsável por endereçar os dispositivos de rede.
